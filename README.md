Setup
1. Create a Supabase project.
2. Open `supabase/schema.sql` in Supabase SQL Editor and run it.
3. Open `js/supabase.js` and replace `SUPABASE_URL` and `SUPABASE_ANON_KEY`.
4. Enable Email auth and Google OAuth in Supabase Auth if you want Google login.
5. Serve locally with VS Code Live Server or any static server.

Do not store raw passwords. Store account hints, recovery info, usernames and notes only.
